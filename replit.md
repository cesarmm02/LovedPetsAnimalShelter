# Loved Pets Animal Shelter

## Overview
This is a static HTML/CSS website for Loved Pets Animal Shelter - an animal rescue organization. The site presents their philosophy and mission to help animals find loving forever homes.

## Project Structure
- `index.html` - Main HTML page
- `style.css` - Styling for the website
- `lovedpts.png` - Logo/favicon
- `main_pets.jpg` - Main image showing pets
- `dog_play.mp4` - Video asset

## Technology Stack
- Static HTML/CSS
- Google Fonts (Quicksand, Merriweather Sans)
- Python HTTP server for local development

## Setup
The project uses Python's built-in HTTP server to serve static files on port 5000 for Replit compatibility.
